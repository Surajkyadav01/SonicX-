🎵 SonicX - Music Streaming Web App
SonicX is a sleek and modern music player web application designed with a focus on user experience and mobile-first responsiveness. Inspired by the Spotify UI, it allows users to browse favorite tracks and search for music within a curated local library.
🚀 Features
Premium UI/UX: Dark-themed interface with neon glow effects and glassmorphism elements.
Local Search Engine: Instant "Suggest-as-you-type" search functionality that highlights songs from the internal library.
Sticky Music Player: A fully functional bottom player with Play/Pause, song details, and a 'Close' feature to manage the view.
Mobile Optimized: Specifically designed to look and feel like a native mobile app on small screens.
Dynamic Backgrounds: Deep charcoal gradients and interactive hover states for a professional feel.
🛠️ Technologies Used
HTML5: Structured the semantic layout including the grid-based song displays.
CSS3: Custom styling featuring Flexbox, CSS Grid, Neon Glow animations, and Responsive Design.
JavaScript (ES6): Logic for audio playback, DOM manipulation for real-time search, and state management for the player.
Font Awesome: Integrated for high-quality professional icons.
